# PHLang Examples

## hello.ph
```
# hello.ph
print "Hello, PHLang!";
```

## math.ph
```
x = 1 + 2 * (3 + 4);
print x;
```

## loops.ph
```
i = 0;
while (i < 5) {
  print i;
  i = i + 1;
}
```

These examples are intentionally simple to help you test each component as it's implemented.
